# Flashcardアプリ開発 作業まとめ

## 1. プロジェクト初期化
- Vite + Vue 3 + TypeScript プロジェクト作成
- Pinia, Vuetify, Firebase, Vue Router を導入
- .env.local でFirebase設定

## 2. Firebase セットアップ
- Firebase Consoleで新規プロジェクト作成
- Firestore Database有効化
- Authentication（メール/パスワード）有効化
- サービスアカウントキー取得
- Firestoreルールを一時的に開放

## 3. データ構造・テストデータ
- Firestoreに `categories`, `words` コレクション作成
- seedData.js でテストデータ一括投入

## 4. フロントエンド実装
- src/firebase.ts: Firebase初期化
- src/stores/auth.ts: 認証ストア（ログイン/サインアップ/状態監視）
- src/stores/flashcard.ts: カテゴリ・単語・進捗管理ストア
- src/components/LoginPage.vue: ログイン画面
- src/components/CategoryList.vue: 問題セット一覧
- src/components/QuizScreen.vue: 出題画面
- src/components/ProgressDashboard.vue: 進捗表示
- src/router/index.ts: ルーティング設定
- App.vue: レイアウト・ナビゲーション

## 5. デバッグ・エラー対応
- Firestoreルールの反映遅延・公開ボタン問題
- 認証状態・Firestoreクエリのデバッグ出力追加
- permission-deniedエラーの詳細調査

## 6. GitHubへのPush
- すべての作業内容をmainブランチに初回コミット＆Push
